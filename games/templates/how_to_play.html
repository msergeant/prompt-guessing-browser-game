How to play

You will be presented with a weird picture that was generated by AI using a prompt.

You will then make a guess as to what prompt was used to generate such a picture.

An example picture might be:

For a prompt like this, you might guess Avocado burger or Stack of veggie burgers.

The correct answer is of course Frog Sandwich!

Then you will see every other player's guess as well as the correct prompt.

You will vote for which prompt you think is correct.

Each vote for the correct prompt is worth 2 points and each guess that receives a vote is worth 1 point.

A game consists of 5 rounds of prompt guessing and voting. Whoever has the highest score after 5 rounds is the winner!
